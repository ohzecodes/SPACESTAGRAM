# SPACESTAGRAM

## API key

Please note that if you are running locally, you must use your own API key.
first generate your key

To do this simply run:

`npm run key`

Enter the api key and you're good to go.

## Running code

First run:

`npm i` to install all neccasary packages in the project (Just do it the first time )

You need two terminal windows over here

1. In the first window run:

then run

`npm run dev` to start the development server

2. In the other window run :

`npm run watch`
to initalize webpack and watch for changes
